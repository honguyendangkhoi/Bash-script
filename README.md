# Bash Scripting Collection 🐧

A repository containing various Bash scripts developed to practice Linux command-line (CLI) skills, automate system tasks, and build foundational programming logic. 

## 📂 Contents Overview

The scripts in this collection generally fall into two main categories:

### 1. System Administration & Automation
Practical scripts for daily Linux environments.
* **`backup.sh`**: A script designed to automate the backup process of files or directories.
* **`my_crontab.txt`**: Configurations for scheduling automated tasks using Linux cron jobs.

### 2. Logic, Algorithms & Data Processing
Scripts built to practice loops, conditions, text manipulation, and algorithmic thinking in Bash.
* **Algorithms:** `binary-search.sh`, `linear-search.sh`
* **Math & Logic:** `leap_year.sh`, `sochansole.sh` (Even/Odd checker)
* **Text & String Manipulation:** `reverse.sh`, `RNA.sh`, `raindrops.sh`, `two-fer.sh`

## 🛠️ Key Skills Demonstrated

- Linux CLI navigation and file manipulation.
- Bash scripting syntax (variables, loops, conditional statements).
- Task scheduling (cron) and basic system backup strategies.
- Implementing standard algorithms directly in the shell.
